# HexaHome-QA-Testing

Comprehensive QA testing of the HexaHome real estate platform including functional testing, bug reporting, performance analysis, SEO validation, and mobile application testing.


## Project Overview

This project contains end-to-end QA testing performed on the HexaHome real estate platform.

### Scope of Testing
- Functional Testing
- Mobile Application Testing
- Performance Testing
- SEO Testing
- Bug Reporting
- Security Recommendations

### Testing Results
- 80+ Test Cases Executed
- 16 Bugs Identified
- 95 SEO Queries Evaluated
- Web & Mobile Testing Completed
- Performance Analysis Conducted Using Google Lighthouse
